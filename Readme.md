# Simple Linear Regression - Sales Prediction

# Objective
To study the relationship between TV advertising expenditure and sales using Simple Linear Regression.

# Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-Learn

# Steps Performed
1. Loaded and explored the Advertising dataset.
2. Split data into training and testing sets.
3. Trained a Simple Linear Regression model.
4. Predicted sales values.
5. Evaluated the model using MSE and R² Score.
6. Visualized results using graphs.

# Results
- Mean Squared Error (MSE): 6.1011
- R² Score: 0.8026

# Conclusion
The model showed a strong positive relationship between TV advertising and sales, explaining approximately 80% of the variation in sales.
